# ⚽ 1vs1チャレンジ

カメラであなたの体の動きを検知して、フェイントでバーチャルDFを釣り、逆を取って突破する1対1トレーニングアプリ。

🎮 **プレイする → https://hkfootballacademy2026-glitch.github.io/1vs1challenge/**

## モード

| モード | 内容 |
|---|---|
| ⚽ ボールあり | 実際のボールを足元に置いてプレー。シザース／キックフェイントでDFを釣り、逆へ持ち出して突破。3ラウンドごとにトータップ課題。 |
| 🎮 ボール無し（仮想ボール） | 画面上の仮想ボールを体の動きで操作。突破成功で仮想ボールがゴールへ！室内・省スペースOK。 |

## 遊び方

1. スマホを立てかける（腰〜胸の高さ推奨）
2. 2歩さがって全身が映る位置に立つ
3. DFが間合いに入ったらフェイント！
   - **シザース**：左右に速く体を揺さぶる
   - **キックフェイント**：大きく振って→ピタッと止める
4. 鋭いフェイント（★評価）ほどDFが食いつく
5. DFが飛んだら**逆方向へ**体ごと抜けて突破！

縦画面・横画面どちらにも対応。音声コーチ（「右だ！」「左だ！」）と振動フィードバック付き。

## 動作環境

- スマホ / PC のモダンブラウザ（Safari, Chrome など）
- カメラ許可が必要（映像は端末内で処理され、外部送信されません）
- 明るい場所・動きの少ない背景だと判定が安定します

## 技術

- 単一HTMLファイル、依存ライブラリなし
- カメラのフレーム差分による動き検知（左右ゾーン＋バーストパターンでフェイント判別）
- Web Speech API（音声コーチ）/ Vibration API
